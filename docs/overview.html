4 Overview

This document presents a table that defines the stochastic transitions for each term in the following system of difference equations. Each rate \(r\) is converted to a transition probability using

\[ P = 1 - \exp(-r) \]

and the number of individuals undergoing that transition is modeled by a binomial draw. For example, the recovery process in compartment \(I_{jt}\) is given by

\[ \Delta R \sim \text{Binom}(I_{jt}, 1-\exp(-\gamma)). \]

The system of difference equations is:

[ \[\begin{aligned} S_{j,t+1} &= S_{jt} + b_{jt} N_{jt} + \varepsilon R_{jt} + \big( \omega_1 V_{1,jt} + \omega_2 V_{2,jt} \big) - \nu_{1,jt} S_{jt} - \big( \Lambda^{S}_{j,t+1} - \Psi^S_{j,t+1} \big) - d_{jt} S_{jt}\\[11pt] V_{1,jt+1} &= V_{1,jt} + \nu_{1,jt} S_{jt} - \omega_1 V_{1,jt} - \big( \Lambda^{V_1}_{j,t+1} - \Psi^{V_1}_{j,t+1} \big) - d_{jt} V_{1,jt}\\[11pt] V_{2,jt+1} &= V_{2,jt} + \nu_{2,jt} V_{1,jt} - \omega_2 V_{2,jt} - \big( \Lambda^{V_2}_{j,t+1} - \Psi^{V_2}_{j,t+1} \big) - d_{jt} V_{2,jt}\\[11pt] E_{j,t+1} &= E_{jt} + \big( \Lambda_{j,t+1} + \Psi_{j,t+1}\big) - \iota E_{jt} - d_{jt} E_{jt}\\[11pt] I_{1,jt+1} &= I_{1,jt} + \sigma\,\iota\,E_{jt} - \gamma_1 I_{1,jt} - \mu_j I_{1,jt} - d_{jt} I_{1,jt}\\[11pt] I_{2,jt+1} &= I_{2,jt} + (1-\sigma)\,\iota\,E_{jt} - \gamma_2 I_{2,jt} - d_{jt} I_{1,jt}\\[11pt] W_{j,t+1} &= W_{jt} + \big( \zeta_1 I_{1,jt} + \zeta_2 I_{2,jt} \big) - \delta_{jt} W_{jt}\\[11pt] R_{j,t+1} &= R_{jt} + \big( \gamma_1 I_{1,jt} + \gamma_2 I_{2,jt} \big) - \varepsilon R_{jt} - d_{jt} R_{jt} \end{aligned}\]

] \tag{4.1}
